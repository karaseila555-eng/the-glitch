# The glitch

**The Zeglet** is a cybersecurity and web analysis tool designed to gather information about websites and their infrastructure.

The tool can be used to discover important information about websites, such as:

* Open ports
* Server information
* Basic network and protocol details

It also includes features for cloning website resources for analysis purposes, such as:

* HTML
* CSS
* JavaScript
* Database-related structures

Future updates will include additional security testing tools and new features to improve the functionality of the project.

The Zeglet is currently under development, and more tools and improvements will be added in upcoming versions.
